# StudySync AI

StudySync AI is a Streamlit-based student productivity platform.

## Features

- Study Planner
- Exam Planner
- Assignment Tracker
- Notes Repository
- Analytics Dashboard
- Attendance Calculator
- GPA/CGPA Calculator

## Tech Stack

- Python
- Streamlit
- SQLite
- Pandas

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
