# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

* Do not disclose security issues publicly until they have been reviewed.
* Contact the maintainers with details of the vulnerability.
* Include steps to reproduce the issue, affected versions, and potential impact.

## Supported Versions

Currently, only the latest version of this project is supported with security updates.

## Response Process

We will:

1. Acknowledge receipt of the report.
2. Investigate the issue.
3. Provide updates on remediation progress.
4. Release a fix if necessary.

Thank you for helping keep this project secure.
