# StudySync AI User Manual

## Introduction

StudySync AI is a student productivity platform designed to help manage assignments, attendance, exams, notes, and academic progress.

## Features

### Dashboard

View overall academic statistics and upcoming tasks.

### Assignment Tracker

* Add assignments
* Track deadlines
* Mark assignments as completed

### Attendance Manager

* Add attendance records
* Monitor attendance percentage

### GPA Calculator

* Add subjects, grades, and credits
* Calculate GPA

### Exam Planner

* Track upcoming examinations
* View exam schedules

### Notes Repository

* Upload and organize notes

## Running the Application

```bash
pip install -r requirements.txt
streamlit run app.py
```
